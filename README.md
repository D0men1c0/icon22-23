# Installazione
Installare SWI Prolog in base al sistema operativo in uso (**scaricare la versione 8**):

`https://www.swi-prolog.org/download/stable?show=all`


Installare la libreria pyswip per far comunicare Prolog con Python:

`pip install pyswip`
